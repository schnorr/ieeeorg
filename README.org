# ieeeorg
The IEEETran in OrgMode, just type make.

Be aware that this example is compatible with OrgMode 8.3.*. Earlier or later OrgMode versions require minor modifications to the Makefile and/or ieee.org files.

For a detailed explanation on why and how to write reproducible articles using OrgMode, please visit the following [[https://github.com/alegrand/RR_webinars/blob/master/1_replicable_article_laboratory_notebook/index.org][page]].
